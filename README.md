# StreamWall

Mac のデスクトップに、HTTP の動画ストリーム（motion の MJPEG など）を
**壁紙のように垂れ流す**ための自分用 macOS アプリです。

- `http://192.168.1.4:8081` のような URL を指定するだけで表示
- 大きさ自由・複数配置・複数ディスプレイ対応
- 普段は**壁紙レイヤー（他アプリの後ろ）に固定**されるので、ブラウザ等の作業の邪魔になりません
- 配置を変えたいときだけ「**配置モード**」に切り替えると、前面に出て移動・リサイズできます
- ストリームごとに **名前・表示ON/OFF・表示方法（全体表示/切り抜き/引き伸ばし）・不透明度** を設定可能
- URL と配置・設定は自動保存され、次回起動時に復元されます
- **多言語対応**: 英語（既定）・日本語・韓国語・ポルトガル語（自動判定 + 手動切替）

メニューバー常駐アプリ（Dock には出ません）です。

---

## 必要なもの

- macOS 13 (Ventura) 以降
- Xcode または Command Line Tools（`swift` が使えればOK）
  - 入っていなければ: `xcode-select --install`

## ビルドと起動

```bash
./build.sh        # StreamWall.app を生成
open StreamWall.app
```

起動すると、メニューバーに ▶️ のアイコンが出ます。
（初回はストリームが無いので、自動で管理画面が開きます）

> ⚠️ 初回起動時に「ローカルネットワーク上のデバイスへのアクセスを許可しますか？」と
> 聞かれたら **許可** してください。motion などローカルIPのストリーム表示に必要です。

## 使い方

メニューバーの ▶️ アイコンをクリック →

| メニュー | 説明 |
|---|---|
| **管理画面を開く…** | URL の追加・編集・削除、配置モードの切替 |
| **配置モード** | ON にすると窓が前面に出て、移動・リサイズできる。OFF で壁紙レイヤーに固定 |
| **すべて再読み込み** | ストリームを再接続したいとき |
| **終了** | アプリを終了 |

### ストリームの追加

1. 管理画面の URL 欄に `http://192.168.1.4:8081` を入れて「追加」
2. デスクトップ中央に窓が出ます
3. **配置モードを ON** にすると窓が前面に出るので、
   ドラッグで移動・端をドラッグでリサイズ・別のディスプレイへ移動
4. 配置が決まったら **配置モードを OFF**。窓が壁紙レイヤーに固定され、垂れ流し開始

窓の削除は、配置モード中に窓の × ボタンを押すか、管理画面の 🗑 ボタンから。

### ストリームごとの設定（管理画面）

| 項目 | 説明 |
|---|---|
| **名前** | 一覧やウィンドウタイトルに使う任意の名前 |
| **表示** | チェックを外すと、その窓を一時的に非表示にできます |
| **表示方法** | 全体表示（はみ出さない）/ 切り抜き（枠いっぱい）/ 引き伸ばし |
| **不透明度** | スライダーで透け具合を調整 |
| 🔄 / 🗑 | 個別の再読み込み / 削除 |

### 言語

管理画面の右上のメニューで切り替えます。`Auto`（既定）は macOS の言語設定に
追従し、対応言語（英語・日本語・韓国語・ポルトガル語）が無ければ英語になります。

> 💡 URL 欄へはコピー&ペースト（Cmd+V）できます。

## 仕組みのメモ

- 各ストリームは独立したウィンドウで、中身は `WKWebView` の `<img>` で MJPEG を表示しています
- ストリームが切れた場合は自動で再接続を試みます（3秒間隔）
- 通常時のウィンドウ階層はデスクトップ（壁紙）のすぐ上・アイコンより下。
  クリックは透過するので、デスクトップ操作の邪魔をしません
- 設定の保存先: `~/Library/Application Support/StreamWall/streams.json`

## 自動起動させたい場合（任意）

`システム設定 → 一般 → ログイン項目` に `StreamWall.app` を追加すると、
Mac 起動時に自動でストリーミングが始まります。
